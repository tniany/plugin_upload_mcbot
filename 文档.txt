# mcbot 假人压测插件

## 功能描述
基于 API 调用 mcbot 程序进行假人压测的 astrbot 插件，仅管理员可使用。

## 指令说明

### /bot start <服务器地址> <假人数量> [延迟最小值] [延迟最大值] [前缀]
- 功能：启动假人压测
- 参数：
  - 服务器地址：例如 localhost:25565（必填）
  - 假人数量：例如 10（必填）
  - 延迟最小值：单位毫秒，例如 4000（可选，默认值：4000）
  - 延迟最大值：单位毫秒，例如 5000（可选，默认值：5000）
  - 前缀：假人名称前缀，例如 ys_（可选，默认值：ys_）
- 示例：
  - 基础用法：/bot start localhost:25565 10
  - 完整用法：/bot start localhost:25565 10 4000 5000 ys_

### /bot stop
- 功能：停止假人压测
- 示例：/bot stop

## API 文档
API 地址：http://localhost:31545/api
API 文档：http://localhost:31545/api/docs

## 实现原理
- 启动测试：调用 /api/start_test 接口，发送 JSON 格式的测试参数
- 停止测试：调用 /api/stop_test 接口

## 默认配置
- 前缀：ys_
- 插件：1.21.1.jar

## 注意事项
1. 仅管理员可使用此插件
2. 确保 mcbot 程序正在运行，并且 API 服务可用
3. 合理设置假人数量，避免服务器过载
